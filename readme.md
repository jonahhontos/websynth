# Web Synth



### Description

A browser-based subtractive synthesizer using the Web Audio API that allows users to create and share presets.

### Technologies

- Backend
  - Mongo, Express, and Node
- Frontend
  - Web Audio API
  - Angular
  - Qwerty Hancock (jQuery musical keyboard input)
  - Angular Material
